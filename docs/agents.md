# Block70 Agents

Agents are **product** background systems that generate opportunity data (opportunities, scores, market intel). They are **not** the same as how humans or Cursor-style tools run **operations** on this repo (rules, PRs, verification scripts). For **Cursor vs Paperclip-class orchestration**, company-role tooling, and in-repo agent guidance, see the root **[AGENTS.md](../AGENTS.md)** file.

---

Agents are background systems that generate opportunity data.

Alpha Hunter
Scans developer ecosystems for early crypto projects.

Sources:
GitHub
Twitter
Discord

Arbitrage Scanner
Monitors decentralized exchanges for price differences.

Sources:
Jupiter
Raydium
Orca

Miner ROI Agent
Tracks mining hardware ROI and profitability.

Inputs:
hardware cost
electricity cost
token price
network difficulty

Wallet Tracker
Tracks high-performing wallets and alerts when they buy or sell assets.

Sources:
Etherscan
Solscan

Scoring Agent
Scores opportunities based on:
upside
risk
difficulty
liquidity
confidence
freshness
