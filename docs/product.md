# Block70 Alpha Network

Block70 is a crypto opportunity intelligence platform that helps users discover profitable opportunities in crypto.

Primary mission:
Answer the question: “Where is the easiest money in crypto right now?”

The platform identifies and ranks opportunities such as:

• arbitrage opportunities
• mining ROI
• node ROI
• airdrops/testnets
• smart wallet activity
• narrative trends

The system aggregates raw data from blockchain explorers, market data APIs, and social sources, converts it into normalized opportunity objects, scores them using a weighted scoring model, and displays them in a dashboard.

Target users:
crypto traders, crypto investors, and crypto opportunity hunters.

Core product features:
• opportunity dashboard
• opportunity scoring engine
• arbitrage scanner
• miner/node ROI tracker
• wallet activity alerts
• user watchlists
• custom alerts
